# 量化私募业绩数据库 & 交互看板

量化私募基金周度业绩数据的导入、存储与可视化看板平台。从"点睛业绩放送"Excel 周报中解析数据，计算累计净值与动态回撤，最终生成自包含的交互式 HTML 看板。

## 数据概览

| 指标 | 数值 |
|------|------|
| 基金公司 | 138 家 |
| 基金产品 | 370 只 |
| 策略类型 | 7 种 |
| 覆盖周数 | 19 周（2026/01/05 – 2026/05/22）|
| 基准指数 | 6 条（沪深300、中证500、中证1000、中证2000、A500、万得全A）|

### 策略分类

| 策略 | 内部标识 | 对比基准 |
|------|----------|----------|
| 量化选股 | `stock_long` | 中证1000 |
| 市场中性 | `market_neutral` | 基准=0（超额=绝对收益）|
| 500指增 | `index_500` | 中证500 |
| 1000指增 | `index_1000` | 中证1000 |
| 300指增 | `index_300` | 沪深300 |
| 2000指增 | `index_2000` | 中证2000 |
| A500指增 | `index_a500` | A500 |

## 数据管道

### 1. 导入周度数据

```bash
cd backend && python scripts/import_weekly_sqlite.py
```

解析 `data/` 下 19 个 Excel 文件，每个文件含 7 个策略 sheet，列布局因周次略有差异（14–17 列）。脚本自动识别列名，统一入库到 SQLite。

**入库字段：**
- `weekly_return` — 周度收益率
- `weekly_excess` — 周度超额收益
- `ytd_return` — 今年以来收益率
- `ytd_excess` — 今年以来超额收益
- `ytd_drawdown` — YTD 动态回撤（**自动计算**）
- `ytd_excess_drawdown` — YTD 超额动态回撤（**自动计算**）
- `ann_return`, `ann_vol`, `max_drawdown`, `sharpe` — 年化指标
- `rank`, `size_category` — 排名与规模分类

**特殊处理：**
- 市场中性策略 `weekly_excess = weekly_return`（基准为 0）
- 动态回撤公式：`dd = (nav - peak_nav) / peak_nav`（峰值法，年初=1.0 为起点）
- 超额动态回撤同理，基于超额累计净值计算
- 规模分类标准化（`~` → `-`）

### 2. 生成看板

```bash
cd backend && python scripts/rebuild_dashboard.py
```

从 SQLite 读取全量数据，构建三个数据对象（`ABS_DATA`、`EXCESS_DATA`、`EXCESS_DD_DATA`），嵌入 HTML 模板生成 `dashboard.html`。看板文件自包含，无需服务器，浏览器直接打开即可使用。

## 看板功能

支持 **3 个 Tab**，右上角切换：

### 📈 绝对收益

| # | 板块 | 说明 |
|---|------|------|
| ① | 策略走势 | 7 条策略平均累计净值曲线 + 6 条基准指数叠加 |
| ② | 策略净值对比 | 按策略/规模筛选，叠加基准对比，展示前 30 产品 |
| ③ | 周度收益排名 | 按周/策略/规模筛选，含柱状图与百分位 |
| ④ | 今年以来收益排名 | 累计净值与 YTD 收益排名 |
| ⑤ | 单基金对比 | 多选基金叠加基准对比 |
| ⑥ | 累计净值排名矩阵 | 全量基金 × 周的累计净值热力图，支持搜索 |
| ⑦ | 收益排名矩阵 | 各产品在所属策略线内每周的名次变化（色块）|

### 📊 超额收益

| # | 板块 | 说明 |
|---|------|------|
| ① | 策略超额走势 | 6 条策略平均超额曲线（不含量化选股）|
| ② | 产品超额净值对比 | 以超额收益百分比展示 |
| ③ | 周度超额收益排名 | 按周/策略/规模筛选周超额 |
| ④ | 今年以来超额排名 | YTD 超额累计排名 |
| ⑤ | 单基金对比 | 多选基金超额曲线叠图 |
| ⑥ | 超额净值排名矩阵 | 全量基金超额热力图 |
| ⑦ | 超额收益排名矩阵 | 策略线内超额名次变化 |

### 📉 超额回撤

| # | 板块 | 说明 |
|---|------|------|
| ① | 策略超额回撤走势 | 6 条策略平均超额回撤曲线 |
| ② | 产品超额回撤对比 | 按策略/规模筛选 |
| ③ | 最大超额回撤排名 | 今年以来最大超额回撤 |
| ④ | 单基金对比 | 多选基金回撤曲线叠图 |
| ⑤ | 超额回撤矩阵 | 全量基金回撤热力图 |
| ⑥ | 超额回撤排名 | 策略线内回撤名次变化 |

### 交互特性

- **三维筛选**：策略类型 × 规模分类 × 管理人搜索（所有筛选器联动）
- **暗色/亮色双主题**：右上角按钮一键切换，偏好自动保存到 localStorage
- **ECharts 图表**：自适应窗口，tooltip 联动，图例切换
- **表格**：固定表头，列排序，行数自适应高度
